# G1 机器人真机操作说明

## 一、准备工作

### 1. 网卡配置

网卡配置位于：`deploy_real/hardware_config.py` 第 11 行

```python
net_if: str = "eth0"  # 默认为 eth0
```

如果需要修改为其他网卡（如 enp131s0），请修改上述配置。

### 2. 运行程序

```bash
cd /home/unitree/dance_deploy
conda activate g1_deploy  # 激活虚拟环境
python deploy_real/deploy_real.py
```

## 二、操作说明

### 2.1 按键说明

| 按键 | 功能描述 |

| **F1** | 被动模式（阻尼模式） - 机器人进入自由状态 |
| **start** | 位置复位（POS_RESET） - 机器人回到初始站立姿势 |
| **select** | 程序退出（急停） - 机器人安全关闭并进入阻尼状态 |
| **R1 + A** | 行走模式（LOCO） - 机器人进入可移动状态 |
| **R1 + B** | 太极（SKILL_12） - 机器人表演太极动作 |
| **R1 + X** | 空翻（SKILL_2） - 机器人做空翻动作 |
| **R1 + Y** | 郭富城舞蹈（SKILL_7） - 机器人表演郭富城舞蹈 |
| **R2 + A** | 45秒舞蹈（SKILL_11） - 机器人表演45秒舞蹈 |
| **R2 + B** | 大花轿（SKILL_16） - 机器人表演大花轿 |
| **R2 + X** | 舞蹈2（SKILL_14） - 机器人表演舞蹈2 |
| **R2 + Y** | 动物园舞蹈（SKILL_15） - 机器人表演动物园舞蹈 |
| **L1 + A** | 行莲舞蹈1（SKILL_17） - 机器人表演行莲舞蹈1 |
| **L1 + B** | 捡箱子（SKILL_18） - 机器人表演捡箱子动作 |

### 2.2 摇杆控制

| 摇杆 | 功能描述 |
|------|----------|
| **左摇杆 (ly)** | 前后方向移动控制 |
| **左摇杆 (lx)** | 左右方向移动控制（反向） |
| **右摇杆 (rx)** | 转向控制（反向） |

## 三、操作流程

### 3.1 启动程序
```bash
python deploy_real/deploy_real.py
```
程序启动后，会首先进行自检，输出 "UnitreeCppEnv self check passed!" 表示成功。

### 3.2 阻尼模式（被动模式）
按 **F1** 键，机器人进入被动模式（阻尼模式），此时机器人处于自由状态。

### 3.3 位置复位
按 **start** 键，机器人回到初始站立姿势。

### 3.4 开始跳舞/技能

> **重要提示**：在启动程序前，请务必先放开 `deploy_real/deploy_real.py` 文件第 111-128 行的技能注释，确保所有技能功能可用。

1. 首先确保机器人处于站立状态（按 start 键复位）
2. 使用组合键执行对应技能：
   - **R1 + B**  - 太极
   - **R1 + X**  - 空翻
   - **R1 + Y**  - 郭富城舞蹈
   - **R2 + A**  - 45秒舞蹈
   - **R2 + B**  - 大花轿
   - **R2 + X**  - 舞蹈2
   - **R2 + Y**  - 动物园舞蹈
   - **L1 + A**  - 行莲舞蹈1
   - **L1 + B**  - 捡箱子

### 3.5 行走模式
1. 按 **R1 + A** 进入行走模式
2. 使用 **左摇杆** 控制前后左右移动
3. 使用 **右摇杆** 控制转向

### 3.6 急停/关闭程序
按 **select** 键，程序会先发送阻尼命令，然后安全关闭机器人。

也可以按 `Ctrl + C` 中断程序，程序同样会执行安全关闭流程。

---

## 四、注意事项

1. **安全第一**：操作时请确保周围空间足够，避免碰撞
2. **使用前检查**：确保机器人处于安全状态后再执行技能
3. **紧急停止**：随时可以按 `select` 键或 `Ctrl + C` 停止程序
4. **阻尼模式**：在进行任何操作前，建议先熟悉阻尼模式，感受机器人的状态
5. **复位**：执行技能后，建议按 `start` 键让机器人回到初始站立状态
